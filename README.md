# AIDS Clinical Trial Data Analysis using SAS

## Project Overview
This project performs statistical and survival analysis on the AIDS Clinical Trials Group Study 175 dataset using SAS. The objective of the project is to analyze treatment effectiveness, patient characteristics, and survival outcomes using biostatistical methods commonly applied in clinical research and healthcare analytics.

---

## Dataset Information
**Dataset Source:**  
UCI Machine Learning Repository – AIDS Clinical Trials Group Study 175 Dataset

The dataset contains patient-level clinical trial data including:
- Age
- Gender
- Weight
- Treatment groups
- CD4 and CD8 cell counts
- Survival outcome
- Survival time

---

## Objectives
- Perform descriptive statistical analysis on clinical variables
- Compare treatment groups using ANOVA
- Analyze relationships between patient variables
- Build logistic regression models for outcome prediction
- Conduct survival analysis using Kaplan–Meier estimation
- Apply Cox proportional hazards regression
- Visualize clinical data distributions and relationships

---

## Statistical Methods Used
- Descriptive Statistics (`PROC MEANS`)
- Frequency Analysis (`PROC FREQ`)
- ANOVA (`PROC ANOVA`)
- Correlation Analysis (`PROC CORR`)
- Logistic Regression (`PROC LOGISTIC`)
- Kaplan–Meier Survival Analysis (`PROC LIFETEST`)
- Cox Proportional Hazards Model (`PROC PHREG`)
- Data Visualization (`PROC SGPLOT`)
